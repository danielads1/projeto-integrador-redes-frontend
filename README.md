## Tema do Projeto
VidaSaudável - Página web com informações, dicas e calculadora de IMC sobre saúde e bem-estar.

# Objetivo
Desenvolver uma página web integrada a uma infraestrutura de rede simulada no Cisco Packet Tracer.

# Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Cisco Packet Tracer

# Estrutura da Rede
- LAN Clientes: 192.168.1.0/24
- Rede Servidor: 10.0.0.0/24

# Funcionalidades JavaScript
- Dicas aleatórias (DOM + evento + array)
- Calculadora de IMC (cálculo + validação + DOM)
- Barra de progresso animada no resultado do IMC
- Cards expansíveis ao clicar
- Relógio e data em tempo real
